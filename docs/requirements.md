# Todo App 需求文档

## 1. 项目目标

为个人提供一个在线待办事项管理工具，随时记录、追踪任务进度。支持管理员管理系统用户和配置。

## 2. 用户角色

| 角色 | 说明 |
|------|------|
| 超级管理员 | 管理用户、系统配置、查看日志。可重置用户密码、禁用/启用用户 |
| 普通用户 | 注册登录后使用，只能操作自己的数据 |

## 3. 功能清单

### P0 — 必须实现

#### 用户模块
- 用户注册（用户名 + 密码），受注册开关控制
- 用户登录（返回 JWT Token + 是否管理员标识）
- 修改密码（用户自助修改）
- 获取用户信息
- 密码加密存储（bcrypt）

#### 任务模块
- 创建任务（标题、内容、分类、优先级）
- 编辑任务（修改标题、内容、分类、优先级）
- 删除任务
- 切换任务状态（待办 ↔ 已完成）
- 查看任务列表（分页）
- 批量操作（批量完成/取消/删除）

#### 系统模块
- 注册开关（管理员控制是否允许公开注册）

### P1 — 重要功能

#### 分类模块
- 查看分类列表（系统预置 + 用户自定义）
- 新增自定义分类

#### 筛选模块
- 按状态筛选（全部/待办/已完成）
- 按分类筛选
- 按优先级筛选
- 关键词搜索

#### 管理员模块
- 用户列表（搜索、分页）
- 禁用/启用用户
- 重置用户密码
- 删除用户
- 系统配置管理

#### 日志模块
- 操作日志（记录关键操作，管理员可查看）
- 登录日志（记录登录时间/IP，管理员可查看）

### P2 — 加分功能

#### 统计模块
- 任务总数 / 已完成数 / 完成率
- 仪表盘图表展示

#### 数据模块
- 导出任务为 CSV

## 4. 业务规则

| 编号 | 规则 |
|------|------|
| BR-01 | 用户名唯一，长度 3-20 位 |
| BR-02 | 密码长度 6-20 位，bcrypt 加密存储 |
| BR-03 | 任务标题必填，最长 100 字符 |
| BR-03-1 | 任务内容选填，最长 1000 字符 |
| BR-04 | 任务状态：0=待办，1=已完成 |
| BR-05 | 优先级：1=高，2=中，3=低 |
| BR-06 | 分类：每个任务只能属于一个分类 |
| BR-07 | 系统预置分类：工作、生活、学习 |
| BR-08 | 所有接口（注册/登录/检查注册开关除外）需携带 JWT Token |
| BR-09 | 用户只能操作自己的任务和分类 |
| BR-10 | 任务列表默认按创建时间倒序排列 |
| BR-11 | 用户状态：0=禁用，1=启用。禁用用户无法登录 |
| BR-12 | 管理员标识：is_admin=1 为管理员，不可自行修改 |
| BR-13 | 管理员不可禁用/删除自己 |
| BR-14 | 注册时需检查系统配置 allow_register，关闭时返回提示 |
| BR-15 | 管理员接口（/api/v1/admin/*）需验证 is_admin=1 |
| BR-16 | 操作日志自动记录：用户管理操作、系统配置变更、任务批量操作 |
| BR-17 | 登录日志自动记录：登录时间、IP、成功/失败 |

## 5. 页面列表

| 页面 | 路由 | 角色 | 说明 |
|------|------|------|------|
| 登录页 | /login | 公开 | 用户登录表单 |
| 注册页 | /register | 公开 | 用户注册表单（受开关控制） |
| 任务主页 | / | 普通用户 | 任务列表 + 筛选 + 统计概览 |
| 修改密码 | 弹窗 | 普通用户 | 修改自己密码 |
| 用户管理 | /admin/user | 管理员 | 用户列表、禁用/启用、重置密码、删除 |
| 系统设置 | /admin/config | 管理员 | 注册开关等系统配置 |
| 操作日志 | /admin/log/operation | 管理员 | 操作日志列表 |
| 登录日志 | /admin/log/login | 管理员 | 登录日志列表 |

## 6. 非功能性需求

| 维度 | 要求 |
|------|------|
| 认证 | JWT Token，过期时间 24h |
| 安全 | 密码加密，接口鉴权，数据隔离，管理员权限校验 |
| 接口风格 | RESTful |
| 字符编码 | UTF-8 |
| 审计 | 关键操作记录日志 |
