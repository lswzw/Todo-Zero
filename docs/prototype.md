# Todo App 原型设计（文本线框图）

## 页面一：登录页 `/login`

```
┌─────────────────────────────────────────┐
│                                         │
│              📝 Todo App                │
│            个人待办管理平台              │
│                                         │
│         ┌───────────────────┐           │
│         │   用户名           │           │
│         └───────────────────┘           │
│         ┌───────────────────┐           │
│         │   密码             │           │
│         └───────────────────┘           │
│                                         │
│         ┌───────────────────┐           │
│         │      登  录       │           │
│         └───────────────────┘           │
│                                         │
│           还没有账号？去注册 →           │
│                                         │
└─────────────────────────────────────────┘
```

**交互说明：**
- 点击「登录」→ 校验 → 成功跳转主页，失败提示错误
- 点击「去注册」→ 跳转注册页

┌──────────────────────────────────────────────────────────┐

├──────────────────────────────────────────────────────────┤

│  └──────────────────────────────────────────────────┘   │
│                                                          │
│  ┌─ 筛选栏 ─────────────────────────────────────────┐   │
│  │ 状态: [全部▼]  分类: [全部▼]  优先级: [全部▼]     │   │
│  └──────────────────────────────────────────────────┘   │
│                                                          │
│  ┌─ 任务列表 ───────────────────────────────────────┐   │

│  │     2024-01-14                          [编辑][删除]│   │

│  │  ☐ 学习 Go Zero 框架        🔴高  📁学习          │   │

│  └──────────────────────────────────────────────────┘   │
│                                                          │
└──────────────────────────────────────────────────────────┘

**交互说明：**
- 点击任务前的 ☐/☑ → 切换完成/待办状态
- 优先级标识：🔴高 🟡中 🟢低
- 点击「编辑」→ 弹出编辑弹窗
- 点击「删除」→ 确认弹窗 → 删除
- 点击「+ 新增任务」→ 弹出新增弹窗
- 筛选栏变化 → 实时刷新列表
- 搜索框输入 → 实时搜索

## 弹窗：新增/编辑任务

```
┌─────────────────────────────────┐
│  新增任务                  [✕]  │
├─────────────────────────────────┤
│                                 │
│  任务标题 *                     │
│  ┌───────────────────────┐     │
│  │                       │     │
│  └───────────────────────┘     │
│                                 │
│  优先级                        │
│  ┌──────────┐                  │
│  │  中 ▼    │                  │
│  └──────────┘                  │
│                                 │
│  分类                          │
│  ┌──────────┐                  │
│  │  工作 ▼  │                  │
│  └──────────┘                  │
│                                 │
│       [取消]    [确定]         │
│                                 │
└─────────────────────────────────┘
```

**交互说明：**
- 编辑时自动回填原有数据
- 标题为必填项
- 点击「确定」→ 校验 → 保存 → 刷新列表
- 点击「取消」或「✕」→ 关闭弹窗

## 组件交互流程图

```
注册页 ──注册成功──→ 登录页 ──登录成功──→ 任务主页
                                        │
                        ┌───────────────┤
                        ↓               ↓               ↓
                   筛选/搜索        新增/编辑弹窗     删除确认弹窗
                        │               │               │
                        └────刷新列表───┘───────────────┘
```

## 页面响应式说明

- 最小宽度 1024px，居中展示，最大宽度 1200px
- 任务列表主体区域宽度 800px 居中
